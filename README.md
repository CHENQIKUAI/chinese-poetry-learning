# 1. 环境安装
nodejs 
mongodb

# 2. 数据导入
打开database文件夹, 执行脚本mongoimport.sh
```shell
sh mongoimport.sh
```

# 安装nppm包
```shell
cd client
npm install

cd server 
npm install

```

# 启动项目

```shell
cd client 
npm start

cd server
npm start
```
