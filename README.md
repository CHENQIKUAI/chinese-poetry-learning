# 1. 环境安装
Node.js 
MongoDB

# 2. 数据导入
```shell
cd database
sh mongoimport.sh
```

# 3. 安装npm包
```shell
cd client
npm install

cd server 
npm install

```

# 4. 启动项目

```shell
cd client 
npm start

cd server
npm start
```
